# Personal Portfolio Website
### Web Development Internship — Level 1 Task 1 Submission

A premium, modern, and fully responsive personal developer portfolio built entirely with **HTML5** and **CSS3**. This project serves as the milestone submission for **Level 1 Task 1** of my Web Development Internship, showcasing my engineering background, technical skillset, and core deployment repositories.

- **Structure:** Semantic HTML5 (`<header>`, `<nav>`, `<section>`, `<footer>`)
- **Styling:** CSS3 (Flexbox layout routing, Grid distributions, Line-clipping gradients, Scale transitions, Breakpoint media queries)
- **Typography:** Google Fonts Integration (Poppins)
