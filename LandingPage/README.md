# LEVEL 1 TASK 2 - Landing Page

This is a responsive Food Delivery Landing Page created as part of the CODSOFT Web Development Internship using HTML and CSS.

---

## 🌐 Project Description

A modern and attractive landing page designed for a food delivery service. It demonstrates proper use of HTML structure and CSS styling including layout, spacing, alignment, and responsiveness.

---

## ✨ Features

- Responsive Navigation Bar
- Hero Section with Heading and Button
- Food Menu Section using Cards (Grid Layout)
- Services Section
- About Section
- Footer Section
- Clean UI with Colorful Design
- Mobile Responsive Layout
